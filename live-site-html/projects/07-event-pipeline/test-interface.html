🔍 What Happens Behind the Scenes:

  • S3 Upload: File uploaded to project-7-serverless-event-upload bucket
  • Lambda Trigger: S3 event automatically invokes processing function
  • Metadata Extraction: Lambda analyzes file and extracts information
  • DynamoDB Storage: Metadata stored in project-7-image-metadata table
  • SNS Notification: Email sent with processing confirmation
