Website Project

This project showcases the complete development of my personal portfolio website using AWS services. The architecture includes CloudFront CDN, S3 static hosting, Route 53 DNS, ACM SSL/TLS, and CI/CD via GitHub Actions with OIDC federation.

Technologies Used: HTML, CSS, JavaScript, AWS S3, CloudFront, Route 53, ACM, GitHub Actions, IAM OIDC
