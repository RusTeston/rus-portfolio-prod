  • Secure: IAM roles with least privilege access

🔄 How It Works

  1. Developer commits code to CodeCommit repository
  2. CodePipeline automatically detects the change
  3. CodeBuild runs automated tests and builds artifacts
  4. If tests pass, code is deployed to S3 static website
  5. Website is updated automatically - no manual intervention!
